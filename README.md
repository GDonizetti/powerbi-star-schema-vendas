# powerbi-star-schema-vendas
# Modelagem e Transformação de Dados com DAX no Power BI

Este projeto demonstra o processo de modelagem dimensional utilizando Power BI a partir da base Financial Sample.

O objetivo foi transformar a tabela original em um modelo dimensional utilizando o padrão Star Schema.

Foram criadas tabelas de dimensão e uma tabela fato utilizando ferramentas de transformação do Power Query e funções DAX.

Tabelas criadas:

- D_Produtos
- D_Produtos_Detalhes
- D_Descontos
- D_Calendario
- F_Vendas

O modelo permite organizar os dados de vendas de forma estruturada para análises em cenários de Business Intelligence.
